Welcome to Tai languages.

Documentations and solutions for Unicode support for Tai languages of Assam including Tai Ahom, Tai Aiton and Tai Phake. The first phase of the project will develop Unicode fonts and keyboard layouts suitable for the Tai languages of Assam.

Locale, collation and web typography support will be developed in a second phase.

Tai fonts

Two Tai Aiton Unicode fonts are under development. The first is a Unicode version of the legacy Tai Aiton font developed by Stephen Morey. The second is an adaption of Keith Stribley's font ThanLwin. Both fonts will contain OpenType and Graphite support. Once Opentype and Graphite support is finalised, we will look at adding AAT support for OSX and iOS.

The Tai Ahom and Tai Phake fonts are based on legacy fonts developed by Stephen Morey.

Keyboard layouts

Tai Ahom, Tai Aiton and Tai Phake keyoard layouts will be implemented using a number of different input frameworks. It is intended to provide solutions for Windows, OSX, Linux, Android and iOS.
